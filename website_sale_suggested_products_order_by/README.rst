======================================================
eCommerce: Static Order for Cart Suggested Accessories
======================================================
* Order the suggested accessories based on _order instead of a randomized order

Configuration
=============
* None needed

Usage
=====
* Add to cart a product that has some suggested accessories defined.
* Go to cart. The order does not change if you refresh the page.
