=========================
Website Sale - firstname2
=========================

* This module adds a "Firstname 2" field to the website sale address form during checkout on Odoo's eCommerce platform. It appears only when creating or editing billing addresses, allowing for the capture of additional name details. The entered information is saved to partner.
