==============================
eCommerce Extra steps settings
==============================

- Allow hiding/showing items in extra step
- Extra step is renamed to Extra info

Configuration
=============
\-

Usage
=====
Go to Extra info-stage and select Customize on upper left.
Then use options 'Hide attachment', 'Hide extra info' and
'Hide reference' to hide elements.
